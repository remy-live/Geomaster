# Geomaster
A lightweight, touch-friendly geometry drawing tool built with HTML5 Canvas.

Geomaster allows users to construct geometric figures directly in the browser using virtual instruments like a compass, ruler, and protractor. It is optimized for both desktop and touch devices (tablets), making it a perfect tool for classrooms.
